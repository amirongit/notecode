# Algebra 2 Full Course
sets: collections of finite or infinite numbers of unique elements
- `null` (or `empty`): unique set which has zero elements
- `U` (or `M`): universal set which contains all elements under consideration
- singletons: sets which contain one single element

set builder notation: describes sets by stating conditions which are satisfied by their members.<br>
subsets: sets which their elements are present in given sets
- improper: contain all of elements of given sets
- proper: contain some of elements of given sets

number of possible subsets is calculated as `2^n` where `n` is equal to number of elements of given set.<br>
complements: sets which contain elements present in `U` & absent in given sets.<br>
unions: sets which contain all elements of given sets.<br>
intersections: sets which contain common elements among given sets.<br>

repeating decimals: decimals which repeat the same pattern of digits forever; patterns are usually annotated with bars on them.<br>
steps to transform repeating decimals into fractions
1. an equation is written which has one variable on one side & given decimal on the other
2. both sides are multiplied by `10^n` where `n` is equal to number of digits of repeating pattern
3. given decimal is subtracted from both sides
4. resulting equation is solved

compound equations or inequations: made of multiple related equations or inequations
- related with `and`: intersection of solutions is their solution
- related with `or` relation: union of solutions is their solutions

absolute value operation: used to indicates distance between its operand & `0`
- its overall form is `|n| = m`
- if `n >= 0`, `m = n`
- if `m < 0`, `m = -n`

steps to solve equations or inequations which contain absolute values
1. absolute values are isolated
    - in case of inequations, if expression is less than `0`, there aren't any solutions
    - in case of equations, if expression is equal to `0`, there is one solution
2. compound equations or inequations are written using bare values (& their additive inverses) instead absolute values
    - in case of inequations, if absolute value side is less than the other side, relationship is `and` & `or` is used in any other situation
3. resulting compound equation or inequation is solved

restrictions of rational expressions (enforced by domains, ranges & expressions)
- divisions by 0
- non real factors (i.e radicals which have even indices & negative operands)
- negative roots of factors raised to even exponents

functions conform to these restrictions.<br>
function notations: used to define & evalute functions.<br>
arguments: used to plug terms as pre definied variables (or choose first components of ordered pairs) within functions; they are restricted by domains.<br>
dependent variables: variables which their values are calculated based on arguments
- explicit form: isolated & replaceable by `f(x)` where `f` is the function & `x` is its argument
- implicit form: not isolated & just indicates existence of functions

joint variations
- their overall form is `y = k(x^n)(z^n)...`
- `k`, `n` & `m` are constants
- same as direct variations having more factors

ordered triples: three terms (components) within parentheses in the form of (`x`, `y`, `z`).<br>
linear equations in three variables (`x`, `y` & `z`)
- their overall form is `ax + by + cz = k`
- `k` is constant
- `a`, `b` & `c` aren't equal to `0` at the same time

matrices: ordered arrays of elements or matrices; they are named in the form of `n*m` where `n` & `m` are the number of rows & columns respectively.<br>
square matrices: matrices which have the same number of rows & columns.<br>
elementry operations on matrices
- row switching: swapping rows
- row multiplication: multiplication of all elements of one row by non zero numbers
- row addition: addition of reuslt of row multiplication operation to rows

augmented matrices: used to represent systems of linear equations in their standard form.<br>
steps to draw augmented matrices
1. equations are transformed into their standard form
2. absent degrees within equations are added with `0` as their coefficients or exponents
3. equations are written without variables
4. columns are used to determine variable factors
5. a vertical line is drawn between coefficients & constants

row echelon form of matrices: non zero rows are above all zero rows & each leading non zero elements is one column right to the non zero elements above it.<br>
reduced row echelon form of matrices: rows & columns conform to row echelon form & there is only one non zero element in each row which is equal to `1`.<br>

methods to solve systems of linear equations in three variables
- substitution
- elimination
    1. equations are transformed into their standard form
    2. one of the variables is eliminated using combination of two equations
    3. the same variable is eliminated using another combination of two equations
    4. resulting system of linear equations in two variables is solved
    5. resulting solutions are used to solve one of the original equations for the eliminated variable
    6. the three values are written in the form of ordered triples: `(x, y, z)`
- augmented matrix
    1. an augmented matrix is drawn based on equations
    2. resulting matrix is transformed into reduced row echelon form
    3. the three values are written in the form of ordered triples: `(x, y, z)`

functions which are used as operands are replaced by their rational expressions.<br>
degree of terms & rational expressions can be reduced by introduction of new variables which are equal to high degree factors.<br>
high degree polynomials can be reduced using this property; substituted variables are replaced with their original form after operations.<br>
in case of dropped out indices & exponents in radical expressions, result is the base itself if they are odd, otherwise, its absolute value.

imaginary unit
- refered to as `i`
- is equal to `radical(-1)`
- doesn't evalute to any real number
- considered as one of the factors of negative operands of radicals which have even indices

`i^2 = -1`<br>
complex numbers
- their overall form is `a + bi`
- `a` is real & `i` is imaginary unit

before applying any operation on non real radicals, their operands are made positive by division of their operands by imaginary unit & transforming them into complex numbers.

vertices: lowest or highest points in parabolas.<br>
vertex form of quadratic equations: `a(x - h)^2 + k`
- `a != 0`
- `h = -b / 2a` & determines horizontal shift from origin by `|h|` units toward right or left if `h > 0` or `h < 0` respectively
- `k = (4ac - b^2) / 4a` & determines vertical shift from origin by `|k|` units toward up or down if `k > 0` or `k < 0` respectively
- vertex happens at `(h, k)`

steps to solve quadratic inequations
1. inequation is transformed into its standard form
2. inequation is written as equation & solved
3. resulting values are used to break the line numbers into ranges
    - if `discriminant < 0`, one single range is assumed
4. one value is picked from each range & plugged in inequation
5. real numbers within ranges of values which satisfy inequation are considered as solutions
6. ranges are written in the form of sets, interval notaions or both conforming to strictness of original inequation

steps to solve rational inequations
1. inequation is manipulated such that it is between `0` & one single fraction
2. denominator & numerator of fraction is solved for `0`
3. resulting values are used to break real numbers into ranges
4. one value is picked from each range & plugged in inequation
5. real number within ranges of values which satisfy inequation are considered as solutions
6. ranges are written in the form of sets, interval notaions or both conforming to strictness of original inequation
    - values which make denominator be equal to `0` are excluded

one to one functions: relations in which every unique `x` corresponds to one unique `y`.<br>
horizontal line test: determines if functions are one to one by examining if they intersect more than once with any horizontal line.<br>
inverse functions: they have domain & range of given functions as range & domain of themselves respectively
- are only defined for one to one functions
- written as `f^-1` where `-1` is superscript & `f` is given function
- derived by solving equations of given functions for argument variables

exponential functions
- their overall form is `f(x) = a^x`
- `a > 0`
- `a != 1`
- their graphs won't touch x axis
- if `a > 1`, their graphs will rise moving toward right
- if `0 < a < 1`, their graphs will fall moving toward right

logarithm operation: used to indicate the exponent to which if its base is raised will be equal to its operand
- its overall form is `log_n(x) = m` where `n^m = x`
- `n > 0`, `x > 0` & `n != 1`
- `n = 10` if not specified explicitly

```
log_n(n^a) = a
n^log_n(a) = a
log_n(n) = 1
log(1) = 0
log_n(a * b) = log_n(a) + log_n(b)
log_n(a / b) = log_n(a) - log_n(b)
log(a^b) = b * log(a)
```

```
# change of base property
# given m = log_n(x)
=> n^m = x
=> log_e(n^m) = log_e(x)
=> m * log_e(n) = log_e(x)
=> m = log_e(x) / log_e(n)
=> log_n(x) = log_e(x) / log_e(n)
```

logarithmic form of exponential functions are their inverse & vice versa.

euler's number
- refered to as `e`
- approximately equal to `2.71`
- logarithms with base of `e` are also written as `ln` (called natural logarithm) instead of `log_e`

methods to solve exponential equations
- manipulation
    1. both sides are manipulated to share common bases
    2. new equation is written based on exponents on each side
    3. resulting equation is solved
- logarithm
    1. both sides are put in logarithms with common bases
        - natural logarithm is used if `e` is among factors
        - is done after isolation as much as possible
    2. variable is isolated
    3. resulting equation is solved

steps to solve logarithmic equations
1. logarithmic terms are merged
2. equation is written in exponential form
3. resulting equation is solved
